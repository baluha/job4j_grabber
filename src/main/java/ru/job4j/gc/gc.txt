Объект User Занимает приблизительно 72 байта в памяти.
Заголовок + объект 16 байт,
поля: int - 4 байта,
      String - с его полями ~ 48 байт
+ 4 округление для кратности 8ми.
Установил размер кучи 3 мб.
Сбощие мусора запускается, когда количество объектов,
на которые никто не ссылается переваливает 7500 шт.
Пи запуске сборщика мусора "живые объекты"(то есть те,
на которые есть ссылка) маркируются, не живые будут удалены.
У живых изменяется счетчик возраста и они будут перемещаться
в памяти из области eden в s0 и s1 а в последствии и в
old generation.
